# 贴吧目录扫完计划 脚本

请确认已安装 `php8.0`, `php-json`, `php-sqlite`, `php-curl`

## tblist

暂无开源计划

---

## tblite

tblite只包含一、二级目录、贴吧名称、gb2312编码的urlencode编码

- 执行 ```php tbilte_run.php```

---

## tbpages

tblite只包含一、二级目录、二级目录页数

- 执行 ```php tbpages_run.php```

---

## 数据库

使用 sqlite，开箱即用
