## 贴吧目录扫完计划·脚本
---
### tblite
tblite只包含以、二级目录、贴吧名称、gb2312编码的urlencode编码
- 导入 **tblite.sql**
- 修改 **config.php**
- 执行```php tbilte_run.php```
- 可运行```php tblite_add.php --fdir=[一级目录] --sdir=[二级目录] --page=[页码]```单独爬取特定页的贴吧
